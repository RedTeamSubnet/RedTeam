# 3. Development

## Prerequisites

- [**Register your wallet on Subnet**](../../manuals/bittensor/register.md): Ensure you have registered your wallet on the RedTeam subnet 61.

## 1. Clone or download relevant repositories and resources

**OPTION A.** Clone the RedTeam miner repository:

```sh
git clone https://github.com/RedTeamSubnet/miner.git && \
    cd miner
```

**OPTION B.** Download the release source code from:

1. Download archived zip or tar.gz file from [**releases**](https://github.com/RedTeamSubnet/miner/releases).
2. Extract it into the projects directory.
3. Enter into the extracted project directory.
