# 2. Development

## Prerequisites

- [**Register your wallet on Subnet**](../../manuals/bittensor/register.md): Ensure you have registered your wallet on the RedTeam subnet 61.

## 1. Choose challenge to solve

- [**RedTeam subnet - challenges**](./../../challenges/README.md): Review the list of available challenges and select one that aligns with your skills and interests:
    - [**Auto Browser Sniffer Challenge**](./../../challenges/ab_sniffer/README.md):
        - Repo: <https://github.com/RedTeamSubnet/ab-sniffer-challenge>
    - [**Anti-Detect Automation Challenge**](./../../challenges/ada_detection/README.md):
        - Repo: <https://github.com/RedTeamSubnet/ada-detection-challenge>
    - [**Humanize Behavior Challenge**](./../../challenges/humanize_behaviour/README.md):
        - Repo: <https://github.com/RedTeamSubnet/humanize-behaviour-challenge>

## 2. Clone or download relevant repositories and resources

**OPTION A.** Clone the RedTeam miner repository:

```sh
git clone https://github.com/RedTeamSubnet/miner.git && \
    cd miner
```

**OPTION B.** Download the release source code from:

1. Download archived zip or tar.gz file from [**releases**](https://github.com/RedTeamSubnet/miner/releases).
2. Extract it into the projects directory.
3. Enter into the extracted project directory.
