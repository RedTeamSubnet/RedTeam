# 3. Choose Challenge to Solve

You need to choose a challenge from the RedTeam Subnet challenges to work on. Each challenge has its own repository and documentation that outlines the requirements and objectives.

## Prerequisites

- [**2. Setup Your Bittensor Wallet**](./2.setup-wallet.md): Set up your Bittensor wallet to participate in the RedTeam Subnet challenges.

## 3.1. Choose challenge

- [**RedTeam subnet - challenges**](./../../challenges/README.md): Browse the list of active challenges and select one that interests you.
    - [**Auto Browser Sniffer Challenge**](./../../challenges/ab_sniffer/README.md)
    - [**Anti-Detect Automation Challenge**](./../../challenges/ada_detection/README.md)
    - [**Humanize Behavior Challenge**](./../../challenges/humanize_behaviour/README.md)
