# Miner Workflow

1. [**Preparation**](./1.preparation.md): Prepare your workspace, environment and, set up your Bittensor wallet
2. [**Develop Solution**](./2.develop-solution.md): Choose a challenge and develop your solution.
3. [**Build and Publish**](./3.build-and-publish.md): Build your solution as a Docker image, publish it to Docker Hub.
4. [**Submit Commit**](./4.submit-commit.md): Submit your published commit (solution) to the RedTeam Subnet for evaluation.
5. [**Monitor and Reward**](./5.monitor-and-reward.md): Monitor your submission status and claim rewards based on performance.
