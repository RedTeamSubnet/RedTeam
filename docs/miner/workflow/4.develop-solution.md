# 4. Develop Solution for Challenge

## 4.0. Prerequisites

- [**3. Choose Challenge to Solve**](./3.choose-challenge.md)

## 4.1. Setup challenge environment

Follow the specific challenge documentation to set up the environment required for the selected challenge. This may include installing dependencies, configuring settings, and preparing datasets.

### Auto Browser Sniffer Challenge

- Repository: <https://github.com/RedTeamSubnet/ab-sniffer-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/ab-sniffer-challenge.git && \
    cd ab-sniffer-challenge
```

### Anti-Detect Automation Challenge

- Repository: <https://github.com/RedTeamSubnet/ada-detection-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/ada-detection-challenge.git && \
    cd ada-detection-challenge
```

### Humanize Behavior Challenge

- Repository: <https://github.com/RedTeamSubnet/humanize-behaviour-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/humanize-behaviour-challenge.git && \
    cd humanize-behaviour-challenge
```
