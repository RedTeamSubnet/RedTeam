# 4. Develop Solution for Challenge

## Prerequisites

- [**3. Choose Challenge to Solve**](./3.choose-challenge.md)

## 4.1. Setup challenge environment

Follow the specific challenge documentation to set up the environment required for the selected challenge. This may include installing dependencies, configuring settings, and preparing datasets.

**Auto Browser Sniffer Challenge**:

- <https://github.com/RedTeamSubnet/ab-sniffer-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/ab-sniffer-challenge.git && \
    cd ab-sniffer-challenge
```

**Anti-Detect Automation Challenge**:

- <https://github.com/RedTeamSubnet/ada-detection-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/ada-detection-challenge.git && \
    cd ada-detection-challenge
```

**Humanize Behavior Challenge**:

- <https://github.com/RedTeamSubnet/humanize-behaviour-challenge>
- Clone the repository:

```sh
git clone https://github.com/RedTeamSubnet/humanize-behaviour-challenge.git && \
    cd humanize-behaviour-challenge
```

## 4.2. Develop your solution

Start implementing your own solution for the chosen challenge. Refer to the challenge documentation for guidelines, requirements, and evaluation criteria.

Examples:

- **Auto Browser Sniffer Challenge**: <https://github.com/RedTeamSubnet/miner/blob/main/examples/ab_sniffer_v5>
- **Anti-Detect Automation Challenge**: <https://github.com/RedTeamSubnet/miner/blob/main/examples/ada_detection_v1>
- **Humanize Behavior Challenge**: <https://github.com/RedTeamSubnet/miner/tree/main/examples/humanize_behaviour_v5>

## 4.3. Test your solution

Thoroughly test your solution locally to ensure it meets the challenge requirements and performs as expected, before proceeding to packaging and submission.

Test manuals:

- [**Auto Browser Sniffer Challenge**](../../challenges/ab_sniffer/testing_manuals.md)
- [**Anti-Detect Automation Challenge**](../../challenges/ada_detection/testing_manuals.md)
- [**Humanize Behavior Challenge**](../../challenges/humanize_behaviour/testing_manuals.md)
